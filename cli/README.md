# Verdict CLI

Command-line interface for working with verdicts.

## Commands

```bash
verdict replay --producer <name> --from <date> --to <date>
verdict eval --producer <name> --dataset <path>
verdict accuracy --producer <name> --window <duration>
verdict gaming-check --producer <name> --agent <name> --window <duration>
verdict list --status <status> --producer <name> --last <duration>
```
